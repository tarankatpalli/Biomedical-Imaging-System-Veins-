For this part of the project to work you must have taken minimum 1500 (can change number if needed) original pictures and convert those images into masks (binary or weighted images used to isolate, 
highlight, or segment vein patterns from background noise, skin, and surrounding tissue). You must use these two sets of pictures (original + masks) in order to create the weights that you 
will need for vein detection (at least for better accuracy). You don't have to take the 1500 images (or more) and convert them to masks by hand. I just created a simple code that took 
1500 images in under 1.5 minutes and organized them for me. Then I created another simple code to create the masks. 

After you have organized these only THEN can you start training the model.
